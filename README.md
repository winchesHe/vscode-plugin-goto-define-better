# Mixins-helper

辅助`Vue2`开发中的`Mixins`使用

## Feature ✨

- [x] 配置式缓存设计（毫秒级切换体验）

- [x] 跳转 Mixins 定义位置

- [x] Mixins 嵌套解析（Mixins中的Mixins）

- [x] Mixins 内容智能补全

- [x] Mixins 内容高亮显示

- [x] Mixins 高亮自定义配置

- [x] Ts 支持（Ts编写的Mixins）

- [x] Vue-class-component 支持（Js|Ts编写的.vue后缀的Mixins）
